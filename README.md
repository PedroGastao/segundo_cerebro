# Segundo_cerebro
Repositório sobre a criação de um "caderno" para pesquisa usando IA. Foi usado o geminiNotebook alimentado com fonte próprias, feito para fazer pesquisas e tirar duvidas sobre a linguagem de programação Python.

### Material usado para a criação:
https://github.com/arbackes/Livro_Python/blob/main/Aprendendo%20Python%20-%20um%20guia%20b%C3%A1sico%20de%20programa%C3%A7%C3%A3o.pdf

https://memoria.ifrn.edu.br/bitstream/handle/1044/2090/EBOOK%20-%20INTRODU%C3%87%C3%83O%20A%20PYTHON%20(EDITORA%20IFRN).pdf

https://do1.dr-chuck.com/pythonlearn/PT_br/pythonlearn.pdf

O principal objeto do caderno seria para fazer pesquisa para alguém que esta aprendendo a linguagem Python ou ate mesmo consultar para tirar uma duvida tendo uma fonte que você saiba a origem e podendo estar gerando vários matérias com o conteúdo.
